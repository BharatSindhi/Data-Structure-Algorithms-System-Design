🧠 What is Bubble Sort?

Bubble Sort is a simple comparison-based sorting algorithm where:

Adjacent elements are compared

Bigger elements “bubble up” to the end

Repeated passes are made until the array is sorted

✅ Your Code (Reference)
#include <iostream>
using namespace std;

void bubbleSort(int arr[], int n) { // O(n^2)
    for (int i = 0; i < n - 1; i++) {          // outer loop → passes
        for (int j = 0; j < n - i - 1; j++) {  // inner loop → comparisons
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);     // swap if out of order
            }
        }
    }
}

void printArr(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr1[] = {1, 6, 7, 6, 7};
    int n = 5;
    bubbleSort(arr1, n);
    printArr(arr1, n);
    return 0;
}

⚠️ Important Improvement (Optimization)

If no swaps happen in a pass → array is already sorted.

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        if (!swapped) break; // already sorted
    }
}


✔ Best case becomes O(n)

🧠 One-Line Exam / Interview Answer

“Bubble sort repeatedly swaps adjacent elements if they are in the wrong order, pushing the largest element to the end in each pass.”
