🚀 C++ string — COMPLETE GUIDE
🔹 What is string in C++?

string is a dynamic sequence of characters provided by the STL (<string>).
It is safer, easier, and more powerful than C-style character arrays (char[]).

#include <string>
using namespace std;

🔸 Why use string instead of char[]?
string	char[]
Dynamic size	Fixed size
Built-in functions	Manual handling
No \0 headache	Needs null terminator
Safe & clean	Error-prone
🔹 Declaring & Initializing Strings
string s1;                 // empty
string s2 = "hello";
string s3("world");
string s4 = s2 + " " + s3; // concatenation

🔹 Input / Output
Using cin (stops at space)
string s;
cin >> s;

Using getline (reads full line)
string s;
getline(cin, s);


⚠️ If getline follows cin, clear buffer:

cin.ignore();

🔹 Access Characters
string s = "hello";
cout << s[0];     // h
cout << s.at(1);  // e (bounds-checked)

🔹 Length / Size
s.length();
s.size();     // same as length()
s.empty();    // true if empty

🔹 Modifying Strings
Append / Concatenate
s += " world";
s.append("!");

Insert
s.insert(5, "!");

Erase
s.erase(1, 2); // erase 2 chars from index 1

Replace
s.replace(0, 5, "Hi");

🔹 Substring
string s = "programming";
string sub = s.substr(3, 4); // "gram"

🔹 Find / Search
s.find("gram");     // index or string::npos
s.rfind("g");       // last occurrence


Check safely:

if (s.find("abc") != string::npos) { }

🔹 Compare Strings
if (s1 == s2) { }
if (s1 < s2)  { }   // lexicographical


Or:

s1.compare(s2); // 0 equal, <0 smaller, >0 larger

🔹 Iterating Over a String
Range-based
for (char c : s) cout << c;

Index-based
for (int i = 0; i < s.size(); i++) cout << s[i];

🔹 Sorting a String
#include <algorithm>
sort(s.begin(), s.end());


Reverse:

reverse(s.begin(), s.end());

🔹 Convert Case (Lower / Upper)
#include <cctype>

for (char &c : s) {
    c = tolower(c); // or toupper(c)
}

🔹 String ↔ Number Conversion
String → Integer / Double
int x = stoi("123");
long y = stol("12345");
double d = stod("3.14");

Number → String
string s = to_string(123);

🔹 Remove Spaces (Common Interview Task)
string res = "";
for (char c : s) {
    if (c != ' ') res += c;
}

🔹 Check Palindrome
bool isPalindrome(string s) {
    int l = 0, r = s.size() - 1;
    while (l < r) {
        if (s[l] != s[r]) return false;
        l++; r--;
    }
    return true;
}

🔹 Frequency Count (Very Important)
vector<int> freq(26, 0);
for (char c : s) {
    freq[c - 'a']++;
}

🔹 Split String (By Space)
#include <sstream>

vector<string> words;
stringstream ss(s);
string word;

while (ss >> word) {
    words.push_back(word);
}

🔹 C-String Conversion (Rare but Asked)
const char* cstr = s.c_str();

⏱ Time Complexity (Quick View)
Operation	Time
Access s[i]	O(1)
Append	O(1) amortized
Insert / Erase	O(n)
Find	O(n)
Substring	O(n)
