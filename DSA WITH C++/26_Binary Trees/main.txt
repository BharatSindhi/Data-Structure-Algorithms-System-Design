🌳 BINARY TREE (DSA)
✅ Definition

A Binary Tree is a non-linear data structure in which:

Each node has at most two children

The children are called:

Left child

Right child

📌 Each node contains:

Data

Left pointer

Right pointer

🧠 Basic Terminology (Very Important)
Term	Meaning
Node	Each element of tree
Root	Topmost node
Parent	Node having children
Child	Node below a parent
Leaf	Node with no children
Height	Longest path from node to leaf
Depth	Distance from root
Subtree	Tree formed by any node
⭐ Structure of a Binary Tree Node
struct Node {
    int data;
    Node* left;
    Node* right;
};

⭐ Types of Binary Trees
🔹 1. Full Binary Tree

Every node has 0 or 2 children.

🔹 2. Perfect Binary Tree

All internal nodes have 2 children

All leaf nodes are at the same level

📌 Nodes = 2^h − 1

🔹 3. Complete Binary Tree

All levels filled except possibly last

Last level filled from left to right

📌 Used in Heap

🔹 4. Skewed Binary Tree

All nodes have only one child

Left-skewed or Right-skewed

🔹 5. Balanced Binary Tree

Height difference between left and right subtree ≤ 1

Example: AVL Tree

⭐ Binary Tree Traversals (VERY IMPORTANT)

Traversal means visiting all nodes.

🔹 1. Inorder Traversal (L → Root → R)
void inorder(Node* root) {
    if(root == NULL) return;
    inorder(root->left);
    cout << root->data << " ";
    inorder(root->right);
}


📌 In BST, inorder gives sorted order

🔹 2. Preorder Traversal (Root → L → R)
void preorder(Node* root) {
    if(root == NULL) return;
    cout << root->data << " ";
    preorder(root->left);
    preorder(root->right);
}


📌 Used to copy tree

🔹 3. Postorder Traversal (L → R → Root)
void postorder(Node* root) {
    if(root == NULL) return;
    postorder(root->left);
    postorder(root->right);
    cout << root->data << " ";
}


📌 Used to delete tree

🔹 4. Level Order Traversal (BFS)
void levelOrder(Node* root) {
    if(!root) return;
    queue<Node*> q;
    q.push(root);

    while(!q.empty()) {
        Node* temp = q.front();
        q.pop();
        cout << temp->data << " ";

        if(temp->left) q.push(temp->left);
        if(temp->right) q.push(temp->right);
    }
}

⭐ Binary Tree Example
        1
       / \
      2   3
     / \
    4   5

Traversal	Output
Inorder	4 2 5 1 3
Preorder	1 2 4 5 3
Postorder	4 5 2 3 1
Level Order	1 2 3 4 5
⭐ Time & Space Complexity
Traversals

Time: O(n) (visit every node)

Space: O(h) (recursion stack)

Worst case skewed → O(n)

Balanced → O(log n)

⭐ Binary Tree vs Binary Search Tree (BST)
Binary Tree	BST
No order	Left < Root < Right
Traversal random	Inorder sorted
Search O(n)	Search O(log n) (balanced)

⭐ Short Exam Answer

"A binary tree is a hierarchical data structure in which each node has at most two children called left and right child. It supports various traversals such as inorder, preorder, postorder, and level order."
