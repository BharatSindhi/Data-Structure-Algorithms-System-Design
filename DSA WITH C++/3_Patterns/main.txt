🔑 Why Pattern Problems Matter

Build logical thinking

Master nested loops

Understand rows, columns, spaces

Prepare you for matrix & recursion problems

🧠 UNIVERSAL APPROACH (VERY IMPORTANT)
Step 1: Observe the Pattern Carefully

Ask:

How many rows?

How many columns?

What changes in each row?

👉 Always think row-wise, not whole pattern at once.

Step 2: Count Rows = Outer Loop

Most patterns:

for(int i = 1; i <= n; i++)   // rows

Step 3: Find Columns = Inner Loop

Inside each row:

Stars / numbers / characters

Spaces (very important!)

Step 4: Decide Printing Rule

For every column:

Print *

or print number

or print space " "

Step 5: Dry Run (MOST IMPORTANT)

Write on paper:

Row 1 → ?
Row 2 → ?
Row 3 → ?

🟢 TYPE 1: BASIC STAR PATTERNS
⭐ Pattern 1: Square Pattern
* * * *
* * * *
* * * *
* * * *

Thinking

Rows = n

Columns = n

Always print *

C++ Code
#include <iostream>
using namespace std;

int main() {
    int n = 4;
    for(int i = 1; i <= n; i++) {
        for(int j = 1; j <= n; j++) {
            cout << "* ";
        }
        cout << endl;
    }
}

🟢 TYPE 2: RIGHT ANGLED TRIANGLE
⭐ Pattern 2

Thinking
Row (i)	Stars
1	1

4	4

👉 Stars = row number

Code
int n = 4;
for(int i = 1; i <= n; i++) {
    for(int j = 1; j <= i; j++) {
        cout << "* ";
    }
    cout << endl;
}

🟢 TYPE 3: INVERTED TRIANGLE
⭐ Pattern 3
* * * *
* * *
* *
*

Thinking

Stars = n - i + 1

Code
int n = 4;
for(int i = 1; i <= n; i++) {
    for(int j = 1; j <= n - i + 1; j++) {
        cout << "* ";
    }
    cout << endl;
}

🟢 TYPE 6: PYRAMID (IMPORTANT 🔥)
⭐ Pattern 6
   *
  * *
 * * *
* * * *

Thinking (KEY CONCEPT)

For each row:

Spaces = n - i

Stars = i

Code
int n = 4;
for(int i = 1; i <= n; i++) {
    for(int space = 1; space <= n - i; space++) {
        cout << " ";
    }
    for(int star = 1; star <= i; star++) {
        cout << "* ";
    }
    cout << endl;
}

🟢 TYPE 7: FULL PYRAMID (ADVANCED)
⭐ Pattern 7
   *
  ***
 *****
*******

Formula

Spaces = n - i

Stars = 2*i - 1

Code
int n = 4;
for(int i = 1; i <= n; i++) {
    for(int space = 1; space <= n - i; space++) {
        cout << " ";
    }
    for(int star = 1; star <= 2*i - 1; star++) {
        cout << "*";
    }
    cout << endl;
}

🧩 COMMON MISTAKES (Avoid These ❌)

❌ Not counting spaces

❌ Thinking pattern as whole, not row-wise

❌ Forgetting endl

❌ Mixing row and column logic

🧠 GOLDEN RULE (EXAM & INTERVIEW)

Row decides everything. Columns obey the row.
