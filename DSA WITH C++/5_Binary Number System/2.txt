🔁 Goal of the Program

Convert a binary number → decimal number

Example:

Binary 1010 → Decimal 10

🧩 COMPLETE CODE (for reference)
#include <iostream>
using namespace std;

int binaryToDec(int biNum){
    int ans = 0, pow = 1;

    while (biNum > 0)
    {
        int rem = biNum % 10;
        ans += rem * pow;

        biNum /= 10;
        pow *= 2;
    }
    
    return ans;
}

int main() {
    cout << "binary to decimal : " << binaryToDec(1010) << endl;    
    return 0;
}

🧠 BASIC CONCEPT (VERY IMPORTANT)
🔹 Binary → Decimal Formula

Each binary digit has a power of 2:

Binary:   1   0   1   0
Position: 8   4   2   1


Decimal value:

= (1×8) + (0×4) + (1×2) + (0×1)
= 10

🔍 FUNCTION EXPLANATION (LINE BY LINE)
1️⃣ Function Definition
int binaryToDec(int biNum)


Takes a binary number (as integer)

Returns its decimal value

2️⃣ Initialize Variables
int ans = 0, pow = 1;

Variable	Meaning
ans	        Final decimal answer
pow	        Current power of 2 (1, 2, 4, 8...)

👉 pow starts from 2⁰ = 1

3️⃣ While Loop
while (biNum > 0)


Loop runs until all binary digits are processed

4️⃣ Extract Last Binary Digit
int rem = biNum % 10;


Gets last digit (0 or 1)

Binary digits are processed right to left

Example:

1010 % 10 = 0

5️⃣ Add to Answer
ans += rem * pow;


Multiply binary digit with its power of 2

Add to final answer

6️⃣ Remove Last Digit
biNum /= 10;


Moves to the next binary digit

7️⃣ Increase Power of 2
pow *= 2;

Step	pow
1st bit	1
2nd bit	2
3rd bit	4
4th bit	8
8️⃣ Return Result
return ans;

🧪 DRY RUN (MOST IMPORTANT 🔥)
Input:
binaryToDec(1010)

Step	biNum	rem	pow	ans
start	1010	-	1	0
1	1010	0	1	0
2	101	1	2	2
3	10	0	4	2
4	1	1	8	10
end	0	-	-	10

✔ Output = 10

🧠 WHY THIS LOGIC WORKS

Binary digits are read right → left

Each digit corresponds to 2⁰, 2¹, 2², ...

%10 extracts digit

pow*=2 moves to next power of 2

⚠️ IMPORTANT LIMITATION (EXAM TIP)

❌ Binary number is stored as int

Leading zeros are lost

Large binary numbers can overflow

✅ INTERVIEW-SAFE VERSION (STRING)
int binaryToDec(string s) {
    int ans = 0, pow = 1;

    for(int i = s.length() - 1; i >= 0; i--) {
        ans += (s[i] - '0') * pow;
        pow *= 2;
    }
    return ans;
}
