A vector in C++ is a dynamic array provided by the Standard Template Library (STL).
It can grow or shrink in size automatically during program execution.

Vector is similar to an array, but its size is not fixed.

⭐ Why Use Vector Instead of Array?
Array	                        Vector
Fixed size	                Dynamic size
Manual memory handling	    Automatic memory management
Less flexible	            More flexible
No built-in functions	    Many built-in functions
⭐ Header File
#include <vector>

⭐ Declaration of Vector
vector<int> v;

⭐ Initialization
vector<int> v = {10, 20, 30, 40};

⭐ Adding Elements
v.push_back(50);   // add at end

⭐ Accessing Elements
Method	Example
Using index	v[0]
Using at()	v.at(1)
First element	v.front()
Last element	v.back()

⭐ Example Program
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v;

    v.push_back(10);
    v.push_back(20);
    v.push_back(30);

    for(int i = 0; i < v.size(); i++) {
        cout << v[i] << " ";
    }
}

⭐ Common Vector Functions
Function	Use
push_back()	Insert element
pop_back()	Remove last element
size()	Number of elements
capacity()	Allocated memory
clear()	Remove all elements
empty()	Check if empty
insert()	Insert at position
erase()	Remove element
resize()	Change size
⭐ Example: insert & erase
v.insert(v.begin() + 1, 100);
v.erase(v.begin() + 2);

⭐ Vector Using Loop (Range-Based)
for(int x : v) {
    cout << x << " ";
}

⭐ 2D Vector (Vector of Vectors)
vector<vector<int>> matrix = {
    {1, 2, 3},
    {4, 5, 6}
};


Access:

cout << matrix[1][2];  // 6

⭐ Memory Behavior of Vector

Vector stores elements in contiguous memory

When capacity is full, vector doubles its size

Old data is copied to new memory

⭐ Advantages of Vector

Dynamic size

Easy insertion/deletion at end

Rich built-in functions

Safe access using at()

⭐ Limitations of Vector

Slower insertion/deletion in middle

Slightly more memory overhead than arrays

⭐ Vector vs Array (Exam Important)
Feature	Array	Vector
Size	Fixed	Dynamic
Memory	Static	Dynamic
STL support	❌	✅
Built-in functions	❌	✅
